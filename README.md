# C# 10 in a Nutshell

Code examples and exercises from the book *C# 10 in a Nutshell* by Joseph Albahari and Eric Lippert.

The book is included in the `Book/` folder.

## Topics Covered

- **CP3**: Classes, OOP, Interfaces, Generics, Enums, Structs
- **CP4**: Delegates, Events, Lambda Expressions

## Running the Project

```bash
cd CSharp-10-in-a-Nutshell
dotnet run
```

## Structure

```
CSharp-10-in-a-Nutshell/
├── CP3/          # Chapter 3 examples
├── CP4/          # Chapter 4 examples
└── Program.cs    # Main entry point
```

## Requirements

- .NET 10.0 SDK (or compatible version)
